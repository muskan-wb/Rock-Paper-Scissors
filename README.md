# ✊✋✌️ Rock Paper Scissors Game

A simple and interactive Rock Paper Scissors game built using HTML, CSS, and JavaScript. The game allows the user to play against the computer with random computer choices, live score tracking, and instant result updates.

---

## 🚀 Features

- Play against the computer
- Random computer move generation
- Live score tracking
- Win, Lose, and Draw detection
- Dynamic result messages
- Interactive hover effects
- Clean and responsive user interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
